ETRI Tag Manager 메뉴얼

1. GT 작업을 할 영상을 불러온다.
2. 객체가 처음 나오는 프레임으로 옮겨가서 box를 친다. 그 이후 박스에 우클릭을 누르면 태그 편집이 나오고, 거기에서 ObjectType을 정한다
(사람 혹은 쓰레기)
2.1. 아래쪽에 썸네일 보기 버튼을 눌러주면 키보드 방향키로 한프레임씩 이동 가능
3. 첫 프레임 박스를 치고, 스페이스를 누르면 local tracker가 돌면서 다음 프레임에 박스 위치를 쳐준다.
3.1. 혹은 오른쪽 상단에 추적 태깅이라는 버튼을 누르면 자동으로 처음부터 끝까지 추적이 돌아간다. 개인적으로는 스페이스가 더 편할듯.
(오른쪽에 < 버튼을 띄우고 나서 스페이스를 눌러야 동작하는 듯. )
4. 마우스로 박스 크기를 보정하고 다시 스페이스 누르면 다음 프레임으로. 
5. 중간중간 위쪽 메뉴에서 Export XML이라고 버튼을 누른다. (저장 기능)
6. 중간에 하던 작업을 불러올 때는 Import XML 버튼을 누르면 기존 작업하던 Tagging을 불러온다.
7. 위쪽 메뉴에서 TagEditor를 누르면 AllTracks에 지금까지 태깅한 물체들의 interval 그리고 박스 좌표들이 나온다. 
7.1. 더블클릭하면 그 위치로 이동하고, 오른쪽 상단에 "<" 마크를 누르면 현재 프레임에 태깅된 객체가 나오고 X를 치면 삭제 가능.
7.2. 삭제 후 다시 TagEditor 창을 껐다가 켜면 Annotation이 변화한 것 확인.

기타.
줌 기능 가능. 
TagEditor에서 ObjectID는 수정 불가. 
프레임 보기 기능을 보면 박스 친 위치가 표시됨. 

각종 버그가 많아서, 자주 ExportXML 하고 ImportXML 하는 것을 반복하는 것을 추천.
기능상의 버그 리포트를 해주시면, Tagging 프로그램 제작 업체에게 수정 요청.
